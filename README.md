# North Beacon Church website

A dependency-free static site designed for GitHub Pages.

## Publish to GitHub Pages

1. Add `index.html`, `styles.css`, `script.js`, and this file to the root of the `nwmdesign/northbeacon` repository.
2. In GitHub, open **Settings → Pages** and choose **Deploy from a branch**.
3. Select the `main` branch and the `/(root)` folder, then save.
4. GitHub will publish the site at `https://nwmdesign.github.io/northbeacon/`.

For the custom domain, add a `CNAME` file containing `www.northbeaconnaz.org`, then set the domain in **Settings → Pages** and update its DNS records with the values GitHub provides.
